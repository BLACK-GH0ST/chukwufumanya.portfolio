Overview

Jagunlabi is a game project developed using Unreal Engine and C++, focused on building core gameplay systems and technical game architecture. The project explores systems such as player interaction, inventory management, weapon handling, and UI integration while emphasizing clean and scalable C++ code.
Through this project, I focus on implementing gameplay mechanics directly in C++, gaining deeper control over performance and system design. Jagunlabi serves as a practical environment where I experiment with game systems, graphics concepts, and gameplay programming techniques used in modern game development.

Developer workflow

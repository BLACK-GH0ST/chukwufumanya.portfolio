Umbra Engine

Lightweight Game Engine & Rendering System

Features

A from-scratch engine built for full control—no black boxes, no bloat. Powered by a modern Direct3D pipeline, it delivers real-time rendering where code becomes visuals instantly. Lightweight, modular, and performance-driven, it’s a developer-first engine focused on clarity, scalability, and mastery.

Rendering The Unseen Experience

Currently building my custom Engine from scratch, developing rendering systems and real-time graphics technology for Umbra Engine. Designed for performance and scalability, the architecture stays lightweight, modular, and highly responsive. Every system is intentionally crafted, making it easy to understand, extend, and optimize. This isn’t just an engine—it’s a learning platform where low-level concepts meet real-time power, and where developers can visualize exactly how their code translates into on-screen results.
